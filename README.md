# KI-Austausch-Teaching-Libraries
Der KI-Austausch für Teaching Libraries bietet eine Plattform, auf der sich
Bibliotheksmitarbeitende über den Umgang mit KI-Tools in unterschiedlichen Lehrangeboten
und über verwandte Themen austauschen können. Der Austausch findet einmal im Quartal statt. Auch
Interessierte aus ähnlichen Fachgebieten sind natürlich willkommen!
